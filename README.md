# meshZup - A Lightweight WhatsApp Client

![meshZup Screenshot](meshzup-screenshot.jpeg)

## Why meshZup?

Had to make a lightweight app, since the official app was taking too much of resources for my liking. Made this app in Rust/Tauri for personal use (with mods) and simply does what it does:

- **Lightweight** and uses less resources
- User agent updated to latest
- oGraph improvements, copy-paste improvements, and handling of media are improved
- **Privacy Screen Feature** (press `Alt+P` to activate)
- And others...
